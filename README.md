# Vedant Kini — Portfolio

A responsive, dependency-free personal website built from Vedant's curriculum vitae.

## Run locally

Open `index.html` directly, or serve the folder from a local web server:

```powershell
python -m http.server 4173
```

Then open `http://localhost:4173`.

## Structure

- `index.html` — semantic site content
- `styles.css` — responsive design system, themes, and animations
- `script.js` — navigation, theme persistence, reveal effects, and canvas orbit
- `assets/` — résumé and favicon

## Customize next

The site is ready for a custom domain, deployment, project detail pages, a portrait, photography, publications, and a writing section.
